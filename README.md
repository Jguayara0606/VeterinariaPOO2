# 🐾 Sistema de Gestión Veterinaria - POO II

Proyecto desarrollado para la **Unidad 3** de Programación Orientada a Objetos II.

## 🚀 Características del Proyecto
Este sistema permite la gestión de mascotas (Perros y Gatos) aplicando conceptos avanzados de Java:

* **Herencia y Polimorfismo:** Clase abstracta `Mascota` y subclases específicas.
* **Patrones de Diseño:** * `Factory Method`: Para la creación segura de objetos.
    * `Singleton`: Para garantizar una única instancia del servicio.
* **Programación Funcional:** Uso de `Streams` y `Lambdas` para filtrado y reportes.
* **Gestión de Errores:** Excepciones personalizadas para validar datos (como edades negativas).

## 🛠️ Tecnologías
* Java JDK 17 o superior.
* IntelliJ IDEA.
* Git / GitHub.
